# Folder Foto Pengurus BOMM

Folder ini berisi foto-foto pengurus BOMM periode 2025-2026.

## Struktur File

Simpan foto pengurus dengan nama file berikut:

### Ketua & Wakil Ketua (Ukuran: 800x800px atau rasio 1:1)
- `ketua.jpg` - Foto Ketua Umum BOMM
- `wakil-ketua.jpg` - Foto Wakil Ketua BOMM

### Divisi/Bidang (Ukuran: 600x600px atau rasio 1:1)
- `sekretaris.jpg` - Foto Sekretaris
- `bendahara.jpg` - Foto Bendahara
- `koordinator-organisasi.jpg` - Foto Koordinator Bidang Organisasi
- `koordinator-psdm.jpg` - Foto Koordinator Bidang PSDM
- `koordinator-humas.jpg` - Foto Koordinator Bidang Hubungan Masyarakat
- `koordinator-medinfo.jpg` - Foto Koordinator Bidang Media & Informasi

## Spesifikasi Foto

### Untuk Ketua & Wakil Ketua:
- **Resolusi**: Minimal 800x800 pixels
- **Format**: JPG/JPEG (lebih ringan dan kompatibel)
- **Ukuran File**: Maksimal 500KB
- **Orientasi**: Portrait (vertikal) atau Square
- **Background**: Bersih, profesional
- **Pencahayaan**: Baik dan jelas

### Untuk Koordinator Divisi:
- **Resolusi**: Minimal 600x600 pixels
- **Format**: JPG/JPEG
- **Ukuran File**: Maksimal 300KB
- **Orientasi**: Portrait atau Square
- **Background**: Bersih, profesional
- **Pencahayaan**: Baik dan jelas

## Tips Foto yang Baik

1. **Komposisi**: Foto close-up atau medium shot (dari dada ke atas)
2. **Ekspresi**: Senyum natural, terlihat ramah dan profesional
3. **Pakaian**: Rapi, formal atau semi-formal
4. **Background**: Polos atau tidak terlalu ramai
5. **Lighting**: Cahaya dari depan, hindari backlight
6. **Focus**: Wajah harus tajam dan jelas

## Cara Mengkompress Foto

Jika foto terlalu besar, gunakan tools online:
- [TinyJPG](https://tinyjpg.com/)
- [Compressor.io](https://compressor.io/)
- [ImageOptim](https://imageoptim.com/) (Mac)

## Placeholder

Jika foto belum tersedia, sistem akan menampilkan icon placeholder otomatis dengan warna background sesuai divisi masing-masing.

## Update Foto

Untuk mengganti foto:
1. Siapkan foto baru dengan spesifikasi di atas
2. Rename file sesuai dengan nama di atas
3. Replace file lama di folder ini
4. Refresh halaman pengurus di website

---

**Note**: Pastikan semua foto memiliki resolusi dan kualitas yang konsisten untuk tampilan yang profesional.
